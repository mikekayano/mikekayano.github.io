COVID-19

The purpose of this project is to gain insights into the COVID-19 pandemic and its impact on diverse countries and populations. I have sourced my data from the Covid Deaths dataset provided by Our World in Data, and am specifically interested in examining statistics related to the number of cases, deaths, and vaccinations across various continents and individual countries. By conducting this analysis, I hope to uncover meaningful insights into the pandemic and how it has affected different regions of the world.

Data Exploration in SQL

This SQL script analyzes Covid-19 data from the "PortfolioProject..CovidDeaths" table. The script has 13 queries that explore the data in different ways. The first few queries select data from the table, filter it based on specific conditions, and order it by certain columns. The next queries calculate the death percentage and infection rate in different countries and continents. The script also includes queries that show Covid-related data for the United States and global numbers. The last few queries use CTE and temporary tables to perform calculations on the data. Finally, the script creates a view to store the data.

Tableau

As part of this project, I have developed a visualization in Tableau that provides a comprehensive and interactive breakdown of COVID-19 data into three categories. The first category, total numbers globally, offers a high-level overview of the pandemic's reach worldwide by displaying the total number of cases and deaths recorded globally, along with a continent-wise breakdown of these figures. This information helps to assess the scale and severity of the pandemic on a global level. The second category, total deaths per continent, examines how COVID-19 has impacted individual continents, displaying the total number of deaths recorded in each continent and comparing death rates between them. This helps gain a deeper understanding of how different regions are affected and how they are coping with the crisis. Finally, the third category, percent of the population infected, provides insights into the spread of the pandemic at a population level by displaying the percentage of the population that has been infected in each continent, giving us a sense of the overall impact of the virus on different populations.

Database Design and
SQL Utilization

In this project, I showcased my expertise in designing a comprehensive database system for McDonald's, emphasizing my skills in ER diagram creation, relational table design, and SQL query formulation. Using MySQL and Visio, I crafted a robust database that reflects the intricate operations of McDonald's, accommodating over 13,000 U.S. restaurants. The project tackled user requirements, encompassing employee roles, customer orders, payments, and product details. The database was realized through a multi-phase approach, starting with a conceptual E-R diagram, followed by logical table design and implementation in MySQL. Query examples demonstrated data retrieval, and I introduced views and stored procedures for efficient data access and analysis. This project underscores my proficiency in transforming complex real-world business scenarios into a structured and efficient database model.

Use of SQL

In this project, I effectively demonstrated the use of SQL through Joins, Views, and Distinct operations to illustrate the database's functionality and its potential use cases. It's important to note that while the data was manually populated, the primary goal was not to extract actionable insights. Instead, the focus was on showcasing the database's structure and its capacity to manage, retrieve, and manipulate data. This project highlights my proficiency in database management and SQL, essential skills for a data analyst, and underscores my ability to create a robust database system tailored to the specific needs of a business scenario.
